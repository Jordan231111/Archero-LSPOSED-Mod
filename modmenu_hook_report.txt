CRITICAL: Native library MUST be stored UNCOMPRESSED in the APK
--------------------------------------------------------------------------------
LSPosed loads native libraries from the APK via base.apk!/lib/arm64-v8a/...
Android's ZipFile/ClassLoader cannot load .so files that are DEFLATED (compressed).
If libarchero_mod.so is compressed (Defl:N), you get:
  java.lang.UnsatisfiedLinkError: ... couldn't find "libarchero_mod.so"

REQUIRED FIX (app/build.gradle):
  packaging {
    jniLibs {
      useLegacyPackaging = false   // Keeps native libs Stored (uncompressed)
    }
  }

VERIFY after every build:
  unzip -l app/build/outputs/apk/debug/app-debug.apk | grep libarchero_mod
  # Must show "Stored" NOT "Defl:N"
